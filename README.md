# Fire-Alarm-System

Simple Fire Alarm System designed using analog electronic components and simulated with Multisim.

## Overview

A simple fire alarm system developed as part of the Electrical Circuits course at Taif University.

The project detects heat using a 10kΩ NTC thermistor. When the temperature increases, the thermistor resistance decreases, activating a transistor that switches on a buzzer alarm. The system provides a low-cost solution for basic fire detection applications.

## Features

- Detects heat using an NTC thermistor
- Automatic buzzer activation
- Analog electronic circuit
- Breadboard implementation
- Multisim simulation
- Low-cost fire alarm prototype

## Hardware

- NTC Thermistor (10kΩ)
- 2N2222 NPN Transistor
- Buzzer
- Resistors
- Breadboard
- Jumper Wires
- 9V Battery

## Software

- Multisim
